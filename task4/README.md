# Playwright Parser API

Проект представляет собой веб-парсер на **Python + Playwright**, который собирает цитаты с сайта и сохраняет их в **PostgreSQL**.  
Также реализован **HTTP API сервер**, через который можно запускать парсинг и получать данные из базы.

---

# Возможности

- веб-парсинг страниц через Playwright
- поддержка пагинации
- опциональная авторизация
- сохранение результатов в PostgreSQL
- HTTP API для запуска парсинга
- получение сохранённых данных через API

---

# Собираемые данные

Для каждой цитаты сохраняются:

| поле | описание |
|-----|-----|
| id | идентификатор записи |
| source_url | URL страницы |
| quote | текст цитаты |
| author | автор |
| tags | теги |
| author_url | ссылка на страницу автора |
| page | номер страницы |
| parsed_at | время парсинга |
