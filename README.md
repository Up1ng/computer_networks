# Компьютерные сети — практические задания (НГУ)

Данный репозиторий содержит практические задания и лабораторные работы по дисциплине **«Компьютерные сети»** в **Новосибирском государственном университете (НГУ)**.

Цель заданий — изучить принципы работы сетевых протоколов, архитектуры сетей, а также получить практические навыки работы с сетевыми инструментами и сетевым программированием.

---

# Содержание репозитория

В директории находятся задачи и лабораторные работы, посвящённые различным темам курса:

- основы сетевой архитектуры
- модель OSI
- модель TCP/IP
- HTTP и HTTPS
- анализ сетевого трафика
- клиент-серверное взаимодействие
- сетевое программирование
- работа с API
- анализ протоколов передачи данных

Каждая задача располагается в отдельной папке и содержит:

- исходный код
- описание задачи
- инструкции по запуску
- примеры использования

---

# Цели курса

В рамках выполнения заданий предполагается:

- понять принципы работы сетевых протоколов
- изучить взаимодействие клиент-сервер
- научиться анализировать сетевой трафик
- реализовать собственные сетевые сервисы
- познакомиться с инструментами для работы с сетью

---

# Используемые технологии

В заданиях могут использоваться следующие технологии и инструменты:

- **Python**
- **HTTP / HTTPS**
- **REST API**
- **curl**
- **Playwright**
- **PostgreSQL**
- **Wireshark**
- **HTTP серверы**
